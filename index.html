Portfolio

This is a repository to showcase some of my projects and track my progress in Data Analytics, Machine Learning and AI topics.

Real-Time Stock Data Streaming and Analysis

This project creates a real-time data streaming architecture using Apache Kafka, Zookeeper, and Python to fetch, process, and analyze stock price data. The system enables efficient real-time data handling, providing a scalable solution for live market analysis and stock price monitoring.

TV AD. Performance Analysis

In this project study advertising and survey data of a client and analyze which networks are the most cost efficient by calculating Cost Per Visitor (CPV), Conversion Rate (CR), and Cost Per Acquisition (CPA) driven by the TV campaign.

AI-Powered Financial Analysis Project

In this project I have developed an AI-driven platform to analyze financial performance for companies like Apple, Tesla, and Microsoft, incorporating a Gemini-powered chatbot for real-time insights.

Fine-tune LLM for Sentiment Analysis

1. I performed full fine-tuning of distilbert-base-uncased on the IMDB dataset for sentiment analysis, optimizing the model to accurately classify movie reviews as positive or negative.

2. I fine-tuned Mistral 7B on the FinGPT dataset using QLoRA to enhance sentiment analysis capabilities for financial text, enabling more accurate interpretations of financial sentiments.

Predict the future trend of an asset

The goal of this project is to predict the future trends of Nvidia, the S&P 500, and Bitcoin using LSTM models trained on historical data. I’ll evaluate prediction accuracy with metrics like RMSE and MAPE, and use indicators like SMA, Support/Resistance, and RSI to improve the forecasts. Ultimately, I aim to assess how LSTM models and these indicators can enhance asset trend prediction.

Exploring Nashville Housing Data

This project enhanced the Nashville Housing dataset by addressing missing values, removing duplicates, and standardizing key variables. These improvements ensure more accurate insights, making the data ready for reliable analysis and predictive modeling, and laying a foundation for better decision-making in real estate forecasting.

Analyze Natural Gas Prices with Time Series

In this project, I applied time series analysis to forecast natural gas prices using models like ARIMA, SARIMA, Exponential Smoothing, and the Transformer-based TimeGPT. I evaluated their performance using accuracy metrics and implemented a chat feature that allows users to query price predictions for specific dates. These enhancements provide a robust framework for accurate forecasting and user-friendly access to predictions.

Credit Risk Analysis

In this project, I examine credit risk using logistic regression and various machine learning models applied to the American Express dataset. Our goal is to identify the most effective model for predicting credit card defaults and to pinpoint the key variables influencing credit risk.
