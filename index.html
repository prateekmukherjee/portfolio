Portfolio

This is a repository to showcase some of my projects and track my progress in Data Analytics, Machine Learning and AI topics.

Enhanced Art Discovery Engine

I created an Art Market Investment Discovery System that treats contemporary art like a financial asset class. It uses predictive modeling and visual analysis to spot undervalued talent. In testing, it didn't just find artists likely to grow; its recommendations saw an average value increase of 245%.

Real-Time Stock Data Streaming and Analysis

I developed a real-time data streaming architecture using Apache Kafka, Zookeeper, and Python to fetch, process, and analyze stock price data. The goal was to enable efficient real-time data handling and provide a scalable solution for live market analysis and stock price monitoring.

TV AD. Performance Analysis

I analyzed advertising and survey data for a client to determine which TV networks were the most cost-effective. I calculated metrics like Cost Per Visitor (CPV), Conversion Rate (CR), and Cost Per Acquisition (CPA) to evaluate the TV campaign's performance.

AI-Powered Financial Analysis Project

I have developed an AI-driven platform to analyze the financial performance of companies like Apple, Tesla, and Microsoft. The platform incorporates a Gemini-powered chatbot to provide real-time insights on factors like revenue, profitability, and growth trends.

Fine-tune LLM for Sentiment Analysis

1. I performed full fine-tuning of the distilbert-base-uncased model on the IMDB dataset to optimize sentiment analysis capabilities for movie reviews.

2. I also fine-tuned the Mistral 7B model on the FinGPT dataset using QLoRA to enhance sentiment analysis for financial text, enabling more accurate interpretations of financial sentiments.

Predict the future trend of an asset

The goal of this project is to use LSTM models trained on historical data to predict future trends for Nvidia, the S&P 500, and Bitcoin. I will evaluate prediction accuracy using metrics like RMSE and MAPE, and incorporate technical indicators like SMA, Support/Resistance, and RSI to improve the forecasts.

Analyze Natural Gas Prices with Time Series

In this project, I applied time series analysis to forecast natural gas prices using models like ARIMA, SARIMA, Exponential Smoothing, and the Transformer-based TimeGPT. I evaluated their performance using accuracy metrics and implemented a chat feature that allows users to query price predictions for specific dates. These enhancements provide a robust framework for accurate forecasting and user-friendly access to predictions.

Credit Risk Analysis

I examined credit risk using logistic regression and various machine learning models applied to the American Express dataset. The goal is to identify the most effective model for predicting credit card defaults and to pinpoint the key variables influencing credit risk.

Using OpenWeatherMap API for Web Scraping

In this project, I demonstrate how to use the OpenWeatherMap API to fetch weather data for a specific city and perform exploratory data analysis on the results.
