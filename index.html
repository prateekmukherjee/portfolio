Portfolio

This is a repository to showcase some of my projects and track my progress in Data Analytics, Machine Learning and AI topics.

Predict the future trend of an asset

Goal is to determine how the use of LSTM models and various indicators to assist in the prediction process for various assets.

AI-Powered Financial Analysis Project

In this project I have developed an AI-driven platform to analyze financial performance for companies like Apple, Tesla, and Microsoft, incorporating a Gemini-powered chatbot for real-time insights.

Real-Time Stock Data Streaming and Analysis

This project builds a real-time data streaming architecture. Apache Kafka, Zookeeper, and Python are used for fetching, processing, and analyzing stock price data.

Exploring Nashville Housing Data

In this project I take raw housing data which is a dump from MSSQL Server, created the required table in PostgreSQL database, imported the csv file and transform it in PostgreSQL to make it more usable for analysis.

TV AD. Performance Analysis

In a recent Data Quality Analysis project, I evaluated the cost-efficiency of TV networks for advertising, focusing on metrics like CPA, CR and Lift. This project demonstrates my ability to drive actionable insights from complex data, ensuring optimized ad strategies and enhanced ROI.

Analyze Natural Gas Prices

This project aims to forecast natural gas prices and provide a user-friendly tool for retrieving price estimates based on specified dates. By analyzing historical price data, it employs three forecasting models—ARIMA, SARIMA, and Exponential Smoothing—to predict future trends and seasonal patterns.
