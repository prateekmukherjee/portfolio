Portfolio

This is a repository to showcase some of my projects and track my progress in Data Analytics, Machine Learning and AI topics.

Real-Time Stock Data Streaming and Analysis

This project builds a real-time data streaming architecture. Apache Kafka, Zookeeper, and Python are used for fetching, processing, and analyzing stock price data.

TV AD. Performance Analysis

In this project study advertising and survey data of a client and analyze which networks are the most cost efficient by calculating Cost Per Visitor (CPV), Conversion Rate (CR), and Cost Per Acquisition (CPA) driven by the TV campaign.

AI-Powered Financial Analysis Project

In this project I have developed an AI-driven platform to analyze financial performance for companies like Apple, Tesla, and Microsoft, incorporating a Gemini-powered chatbot for real-time insights.

Fine-tune LLM for Sentiment Analysis

1. Full finetuning of distilbert-base-uncased on IMDB dataset for sentiment analysis.

2. Fine-tune Mistral 7b on FinGPT dataset designed to assist with sentiment analysis with QLoRA.

Predict the future trend of an asset

Goal is to determine how the use of LSTM models and various indicators to assist in the prediction process for various assets.

Exploring Nashville Housing Data

In this project I take raw housing data which is a dump from MSSQL Server, created the required table in PostgreSQL database, imported the csv file and transform it in PostgreSQL to make it more usable for analysis.

Analyze, Predict Nat_Gas Prices with Time Series

I implement time series analysis of natural gas prices using various forecasting models like ARIMA, SARIMA, Exponential Smoothing and Transformer based time series model TimeGPT. Performance metrics are computed to assess their accuracy. A chat function is implemented to allow users to inquire about specific dates. .

Credit Risk Analysis

In this project, I examine credit risk using logistic regression and various machine learning models applied to the American Express dataset. Our goal is to identify the most effective model for predicting credit card defaults and to pinpoint the key variables influencing credit risk.

In this project I demonstrate how to use the OpenWeatherMap API to fetch weather data for a specific city and perform EDA on the results:.
